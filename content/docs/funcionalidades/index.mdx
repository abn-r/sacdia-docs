SACDIA cuenta con 22 funcionalidades implementadas que cubren la gestion completa de clubes de Conquistadores, Aventureros y Guias Mayores. Esta pagina registra el estado de implementacion de cada modulo a traves de las tres capas del sistema: backend (NestJS), panel admin (Next.js) y app movil (Flutter).

## Resumen de funcionalidades

| Dominio | Estado | Backend | Admin | App |
|---------|--------|---------|-------|-----|
| [Gestion de Clubs](/docs/funcionalidades/gestion-clubs) | IMPLEMENTADO | ClubsModule (14 endpoints) | 3 pages funcionales | club + members + units |
| [Gestion de Seguros](/docs/funcionalidades/gestion-seguros) | IMPLEMENTADO | InsurancesModule (4 endpoints) | CRUD funcional | 3 screens |
