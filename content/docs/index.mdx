SACDIA es el sistema institucional de trazabilidad para los clubes del Ministerio Juvenil Adventista. Registra, preserva y opera la trayectoria de miembros dentro de los programas de Conquistadores, Aventureros y Guías Mayores.

## Qué es SACDIA

SACDIA no es solo un panel administrativo ni una app. Es un sistema institucional que organiza la relación entre miembros, clubes, secciones, procesos formativos, validaciones y niveles de autoridad a lo largo del tiempo.

La unidad de valor principal es la **trayectoria institucional del miembro**. Todo lo demás — secciones, procesos, validaciones, reportes y vistas operativas — existe para sostener, interpretar o proyectar esa trayectoria.

## Componentes del sistema

SACDIA se compone de tres aplicaciones que cooperan sobre una base común:

- **sacdia-backend** — API REST en NestJS con la lógica de negocio, seguridad y contratos operativos
- **sacdia-admin** — Panel web en Next.js para gestión, supervisión y operación administrativa
- **sacdia-app** — App móvil en Flutter para interacción contextual y uso cotidiano
