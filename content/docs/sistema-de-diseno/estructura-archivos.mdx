## Convenciones de Naming

| Tipo | Convencion | Ejemplo |
|------|-----------|---------|
| Componentes | `kebab-case.tsx` | `page-header.tsx` |
| Paginas | `page.tsx` (Next.js convention) | `app/dashboard/page.tsx` |
| Layouts | `layout.tsx` (Next.js convention) | `app/dashboard/layout.tsx` |
| Tipos/interfaces | `PascalCase` | `UserProfile`, `ClubMember` |
| Variables y funciones | `camelCase` | `handleSubmit`, `isLoading` |
| Constantes | `SCREAMING_SNAKE_CASE` | `MAX_FILE_SIZE` |
| CSS custom props | `--kebab-case` | `--sidebar-accent` |

## Donde va cada componente

| Si es... | Va en... |
|----------|----------|
| Componente shadcn/Radix instalado | `components/ui/` |
| Componente reutilizable sin dominio | `components/shared/` |
| Componente de layout (sidebar, header) | `components/layout/` |
| Componente especifico de una feature | `components/[feature]/` |
| Logica de datos o llamadas API | `lib/[dominio]/` |
| Tipos TypeScript | `lib/types/` o co-locado con el feature |
| Hooks personalizados | `lib/hooks/` |
