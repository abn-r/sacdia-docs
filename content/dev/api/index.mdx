## Formato de Respuesta Estándar

Todas las respuestas siguen el mismo envelope:

```json
{
  "status": "success",
  "data": { },
  "meta": {
    "timestamp": "2026-01-29T17:00:00Z",
    "version": "1.0.0",
    "requestId": "uuid"
  }
}
```

Los errores usan:

```json
{
  "status": "error",
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Validation failed",
    "details": [{ "field": "email", "message": "must be a valid email" }]
  }
}
```
