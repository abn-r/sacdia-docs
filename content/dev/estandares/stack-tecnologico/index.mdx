SACDIA es una plataforma full-stack compuesta por tres aplicaciones principales que comparten convenciones, autenticacion y base de datos.

## Resumen del Stack

| Componente | Tecnologia | Lenguaje |
|---|---|---|
| Backend API | NestJS 10.x | TypeScript 5.x |
| Admin Web | Next.js 16 (App Router) | TypeScript 5.x |
| App Movil | Flutter 3.x | Dart 3.3+ |
| Base de Datos | PostgreSQL 15.x (Supabase) | SQL |
| ORM | Prisma 5.x | - |
| Auth | Supabase Auth | - |
| Cache | Redis (Upstash) | - |
| Storage | Supabase Storage / Cloudflare R2 | - |

## Metodologia

**Arquitectura**: Backend REST API + Multiples Clients (Web Admin + Flutter App)

**Metodologia de Desarrollo**: Clean Architecture aplicada en todas las capas

**Package Manager**: pnpm (backend y admin), pub (Flutter)
